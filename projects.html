Project 1

SkillBridge

Staging → Partial Production

SkillBridge is a verified talent-pipeline platform connecting job-ready candidates with employers through skill assessments and verified profiles. As the sole QA engineer on the team, I was responsible for testing core functionality across authentication, assessments, and role-based access.

I built and ran an automated API regression suite of 78 requests across 11 modules using Postman and Newman CLI — covering auth, session handling, settings, assessments, and role-based access guards with both positive and negative test cases. I also executed a manual test pass on the Verified Profile feature against an 11-point acceptance charter, logging defects with severity, reproduction steps, and environment data.

Project 2

ClinSight

MVP Testing

ClinSight is a healthcare AI platform. I joined the QA effort during the MVP stage, focusing on the authentication flow, OTP verification system, and API reliability.

I designed and executed a structured QA test plan covering user signup, OTP verification, password reset, session validation, and API error handling. The plan included entry and exit criteria, a risk-based testing priority framework, and boundary testing for OTP expiration and failed attempt limits.
